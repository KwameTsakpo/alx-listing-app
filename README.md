# ALX Listing App

The **ALX Listing App** is a simplified Airbnb clone project built using **Next.js**, **TypeScript**, and **Tailwind CSS**.  
It helps learners practice building reusable, scalable UI components.

---

## Project Goals

- Create a reusable UI for property listings.
- Use TailwindCSS for responsive styling.
- Apply best practices with Next.js and TypeScript.

---

## Project Structure

| Folder | Description |
|--------|--------------|
| `components/` | Contains reusable UI components (e.g., `Card`, `Button`). |
| `interfaces/` | Holds TypeScript interfaces. |
| `constants/` | Stores constants like API URLs or app name. |
| `public/assets/` | Contains images and icons. |
| `pages/` | Defines all routes for the application. |
